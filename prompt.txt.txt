You are a Senior Full Stack Engineer, Principal Solution Architect, Staff UI/UX Designer, Product Manager, and Technical Lead.

Build a complete Vercel-deployable web application called:

OLAM AGRICONNECT EXPERT PORTAL

The application must be production-quality in architecture and code quality, while remaining lightweight enough for Proof of Concept deployment.

====================================================
PROJECT PURPOSE
===============

The application supports an agricultural WhatsApp chatbot platform used by Olam.

Farmers communicate through WhatsApp.

The chatbot supports:

* Surveys
* Training Modules
* Voice Responses
* Plant Disease Diagnosis

The primary purpose of this dashboard is to allow agricultural experts to review AI-generated plant disease diagnoses and respond to farmers.

===============

Each diagnosis case contains:

{
"caseId": "CASE_001",
"phoneNo": "917085221146",
"imageUrl": "https://...",
"aiResponseFarmer": "...",
"aiResponseDashboard": "...",
"status": "EXPERT_REVIEW_REQUIRED",
"createdAt": "2026-06-18T10:30:00Z"
}

====================================================
INCOMING CASE API
=================

Create an API endpoint:

POST /api/cases

Purpose:

Receive new diagnosis cases from n8n.

Request Example:

{
"caseId": "CASE_001",
"phoneNo": "917085221146",
"imageUrl": "https://...",
"aiResponseFarmer": "Thank you for sharing the image.",
"aiResponseDashboard": "Possible Black Pod Disease",
"status": "EXPERT_REVIEW_REQUIRED",
"createdAt": "2026-06-18T10:30:00Z"
}

Store case data.

Return:

{
"success": true
}
