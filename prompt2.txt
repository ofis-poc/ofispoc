You are building an enterprise Survey Builder for the Olam AgriConnect Expert Portal.

The application is built using:

Next.js App Router
TypeScript
TailwindCSS
PostgreSQL
Existing Survey APIs
Existing multilingual translation API (FreeLLMAPI)

The goal is to replace Google Sheets completely.

This Survey Builder will become the single place where administrators create WhatsApp surveys for farmers.

Overall UX

The flow should be exactly like Google Forms.
