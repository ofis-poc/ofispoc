Disease Confidence: 95%

Expert Review:

### Issue 2: Expert Insights Card

Parse `aiResponseDashboard` before rendering.

Handle both cases:

1. `aiResponseDashboard` is a JSON string
2. `aiResponseDashboard` is already an object

Use safe parsing:

```ts
try {
  parsed = typeof aiResponseDashboard === 'string'
    ? JSON.parse(aiResponseDashboard)
    : aiResponseDashboard;
} catch {
  parsed = null;
}
```

Never allow invalid JSON to crash the page.
